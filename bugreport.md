## Как правильно сообщить о баге

Чтобы мы могли быстро воспроизвести и исправить проблему, пожалуйста, создавайте баг-репорты по следующим правилам.

### 1. Заголовок

Кратко и по делу:

- Плохо: `Помогите!!!`
- Хорошо: `[Web] Краш при сохранении профиля`
- Формат: `[платформа/страница] что ломается + когда`

### 2. Что указать в описании

Пожалуйста, скопируйте и заполните шаблон:

```md
**Окружение:**
- Устройство: (ПК / ноутбук / телефон, модель)
- ОС: (Windows 11, Android 14, iOS 18, Linux и т.д.)
- Браузер / версия приложения: (Chrome 143, Firefox, Safari…)
- Тип аккаунта (если важно): (гость / пользователь / админ)

**Шаги для воспроизведения:**
1. ...
2. ...
3. ...
4. ...

**Ожидаемый результат:**
Что должно происходить.

**Фактический результат:**
Что происходит на самом деле (текст ошибки, некорректное поведение и т.д.).

**Повторяемость:**
Каждый раз / иногда / один раз.

**Важность:**
minor (мешает, но жить можно) /  
major (сильно мешает использовать функционал) /  
blocker (невозможно пользоваться приложением).

**Доп. информация:**
Любые наблюдения, которые могут помочь (например: "если убрать цифры из имени — сохраняется").

**Приложения:**
Скриншоты / видео / логи (если есть).
